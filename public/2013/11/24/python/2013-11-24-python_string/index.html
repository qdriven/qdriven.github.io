浮点数算法：争议和限制

使用计算模块 division

使用python的模块来作除法：from __future__ import division 所有的除法就都是浮点数了:

from __future__ import division
print 5/2  # 2.5 如果不用division，则为2

整除方法：

print divmod(5,2) # 返回元组（2，1）
print divmod(10,3)[0],divmod(10,3)[1],divmod(10,3)
# 结果是：
3 1 (3, 1)

四舍五入 round

四舍五入取位数：

print round(1.2345567, 2)
print round(1.23456677, 3)
print round(10.0 / 3, 4)

结果是：

1.23
1.235
3.3333

以下结果不要惊慌,这是正常的！！！！

round(1.2345,3)
1.234               #应该是：1.235
round(2.235,2)
2.23                #应该是：2.24

优先级

以下是优先级从低到高的顺序：

运算符 描述
lambda Lambda表达式
or 布尔“或”
and 布尔“与”

~x 按位翻转

x.attribute 属性参考

(experession,…) 绑定或元组显示
