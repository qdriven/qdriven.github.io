# 2020 Adventure

2020，测试老鸟准备开始一个新的尝试. 其实没有什么所谓新的尝试，无非原来的自己不够自律，现在准备更加自律一些.
老鸟知道对于大部分普通人来说，都喜欢学习，只是学习和自律是反人性的活动，如果你能完成你定的目标的10%，我觉得已经
非常不错了，如果不信的话，你看看你的书桌上10本书，你好好看过的合起来超过1本书的量了吗?

测试老鸟工作很久，没有什么成绩，刚够养活自己，也没有什么宏伟理想，无非就是觉得工作中被各种各样的事情纠缠，心里一直想
难道没有另外一些方法做同样的事情吗？所以准备把自己的想法和实践留个记号. 

工作中大部分事情，经常会牵扯到产出，流程和效率. 所以所有的记号大部分都围绕着这三点进行。产出由于和公司业务相关，最多也就是一些思考，流程其实各家公司大同小异，而效率确实非常不同。 所以更多的想说说效率的事情，老鸟对于效率的认识是，主要两个方面:
- 沟通
- 工具的使用
- 日常tips
