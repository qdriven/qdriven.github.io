# Python开发环境在MAC的搭建

在MAC上面搭建Python环境相对比较简单，MAC上面默认就安装了Python的语言包，不过呢，MAC的python默认版本是2.*版本，
python的主流趋势就是使用python 3.*，所以下面主要介绍Python3的搭建，主要包括一下几个方面：

1. 安装和配置Python3
2. 如果使用Python3自带的虚拟环境
3. 如何使用pip管理需要的依赖包
4. 一些常见问题

## Python3 MAC 安装

Mac上面安装软件大体方法有如下两种:

1. 直接下载某个安装包
2. 使用**brew** 工具安装

Python3 安装个人推荐可以直接下载安装包安装，安装完毕之后在命令行(terminal)中直接运行:

```sh
python
```

如果显示python的版本还是很2.*，可以尝试使用运行python3:

```sh
python3
```

以上的配置可能会造成的问题是，你如果想用系统级别安装python的依赖包的话，pip就不能直接使用，因为python2 默认不支持pip. 不过记住以下两点,可以直接使用: **python3** 和 **pip3** 命令，这样就完全使用python3的命令了.
